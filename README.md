# Previsão+

Aplicativo para visualizar a temperatura com base na sua localização.

## Iniciar

1. Clone o repositório

   ```bash
   git clone https://github.com/DaviFreitas77/previsao.git
   ```

2. Instalar dependencias

   ```bash
   npm install
   ```
2. Rodar o projeto

   ```bash
   npx expo run:android
   ```
